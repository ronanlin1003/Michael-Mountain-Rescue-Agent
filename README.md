# 🚁 米迦勒高山搜救長程安全代理系統 (Project Michael)

本專案專為極端高山難後勤調度設計，旨在透過 NVIDIA AI 生態系，打造一套兼具「長程自主推理能力」與「剛性法律安全底線」的自主特種搜救代理人（Long Agent），並結合虛擬模擬環境進行無人載具的戰術調度。

## 🌟 核心架構與三大階段藍圖

我們將米迦勒計畫規劃為三個完備的演進階段，目前黑客松階段已成功完成第一階段（PoC 核心骨架與多維度法律攔截驗證）：

### Phase 1：安全與法律防護欄驗證 (現狀已實作)
* **OpenClaw 狀態機骨架 (`main.py`)**：建立核心搜救長程工作流，接收突發山難情境通報並進行任務拆解。
* **NemoClaw 跨領域合規防護欄 (`guardrails.co`)**：獨立於 LLM 大腦之外的剛性安全晶片鎖。針對無人機/機器狗在極端救災時可能引發的「自主腦衝違法行為」，實作無人介入（No-human-in-the-loop）的毫秒級攔截，確保國家安全與司法底線。

### Phase 2：數據感知與 Cosmos 虛擬場域模擬 (未來擴充 - 開發中)
* **NVIDIA Cosmos / Omniverse 數位雙生整合**：在虛擬世界中完整重現台灣玉山主峰與風口地形的 3D 數位雙生場域。利用 Cosmos 模擬高山極端暴風、大雨、低能見度等物理環境。
* **無人機與機器狗模擬訓練 (Sim-to-Real)**：在 Cosmos 虛擬場域中，自主訓練無人機群的「強風盲飛避障演算法」與四足機器狗（如 Unitree）的「碎石坡攀爬生命線調度」。
* **即時氣象與 GPS 軌跡整合**：對接中央氣象署 API 與山友回傳的 `.gpx` 軌跡，將真實世界的數據同步輸入 Cosmos 虛擬指揮所中，動態編排概率搜索區域（Grid Search Area）。

### Phase 3：自主演算與跨載具實體調度 (未來擴充 - 願景)
* **黃金救援時間計算**：結合步速、地形坡度與失溫模型，自主推算失聯山友的黃金搜救時限。
* **多模態戰術編排**：由 Nemotron-3-Super 核心推理模型進行決策，調派透過 Cosmos 驗證過的「特搜無人機群」進行空中搜訊，並指派馱負威而鋼、丹木斯等高山症急救藥品與保暖物資的「特搜機器狗群」挺進盲區。

## 🛡️ NemoClaw 核心防護機制說明

為了防止 OpenClaw 為了達成「救人第一」的任務目標而盲目違法，本系統透過 NemoClaw 建立三道鋼鐵防線：
1. **國安防護欄（《軍事要塞堡壘法》）**：自動攔截並隱蔽（Masking）無人機傳輸中包含軍事管制區的高精度 3D 圖資與影像。
2. **電信防護欄（《電信管理法》）**：嚴格禁止 AI 為了突破通訊盲區而自主解鎖軟體接口超頻，避免無線電蓋台干擾國防與民航求救專用頻段（121.5MHz），保護搜救直升機航管安全。
3. **司法防護欄（刑事訴訟現場保全原則）**：當多模態視覺判定受難者已無生命跡象（DOA）且現場具刑事他殺特徵時，自動鎖定機器狗實體機械臂，轉入保全錄影模式，防止 AI 為了確認身分而污染命案現場。
